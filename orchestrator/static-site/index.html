Teacher Dashboard

Admin interface for managing students, reviewing conversations, setting system prompts, and tracking engagement metrics. Full visibility into what every student is working on.

Student Workspaces

Each student gets their own isolated AI tutor with persistent conversation history, custom system prompts, and in-browser Python code execution (Pyodide).

Why Schools Choose Almanach

Privacy-first education infrastructure. No cloud lock-in. No data harvesting.

🔒

Data Stays On-Premise

Student conversations, code, and progress never leave your server. Run on your hardware, your VPS, or our managed Canadian infrastructure. FERPA-aligned by default.

📦

Isolated Per-Student Agents

Each student's AI runs in its own container. One student's context never leaks to another. Teachers can audit any conversation, any time.

🔑

Bring Your Own API Keys

Use your own keys from OpenAI, Anthropic, Google, or any provider. You control costs and data routing. No middleman taking a cut.

💻

In-Browser Python

Students write and run Python code directly in the chat interface — powered by Pyodide (WebAssembly). No server round-trip for code execution. Matplotlib, NumPy, and more.

⚡

Chromebook-Compatible

Web-based interface works on any device with a browser. No installs, no IT headaches. Students log in and start learning immediately.

🚫

No Vendor Lock-In

Standard containers, open protocols, MIT license. If you want to move your deployment elsewhere, your data and configuration go with you.
